## Authoring conventions

- One Markdown file per article
- Front-matter with title, slug, published date, audience
- Headings start at H2 (the H1 is the title in front-matter)
- Pull quotes use blockquote syntax (`>`)
- Code snippets use triple-backtick with language tag
- Cross-link with relative paths to other docs in this repo
